About

Our game is a platformer it is endless so there is no ending. You travel though the game to get coins to use in the shop (comming soon). You try not to die from the enemy or from falling onto the nails. Collect power ups and get as many coins as possible.

Supported platforms

Currently the game only supports Android. in the future we will support more platforms like ios, and windows. Hopefully we can do this.

Help us

If you would like to help us you can contact us (contact page) and tell us what we should add to make the game more fun. We will consider adding it into the games it can also make the game more fun

game disclaimers
