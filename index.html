about the game

Our game is a free to play platformer it is endless so there is no ending. You travel though the game to get coins to use in the shop (comming soon). You try not to die from the enemy or from falling onto the nails. Collect power ups and get as many coins as possible.

Supported platforms

Currently the game only supports Android. in the future we will support more platforms like ios, and windows. Hopefully we can do this.

game disclaimers

This game only works on 64 bit devices only if your device is 32bit the game will not install no matter what we are working on a 32 bit version for lower end phones. If you really want to play you can always you emulators like bluestacks or nox. Please bear with us while we complete the 32bit port. The game has optimization problems and a few bugs that we are fixing. Read News for updates.
